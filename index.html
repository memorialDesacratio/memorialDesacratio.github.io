🕵️ OSINT

Что такое OSINT?

OSINT (Open Source Intelligence) — разведка по открытым источникам. Это сбор и анализ информации из общедоступных источников.

Основные инструменты

  • Maltego — визуализация связей между данными
  • Shodan — поиск устройств, подключённых к интернету
  • theHarvester — сбор email, поддоменов, IP
  • Recon-ng — фреймворк для веб-разведки
  • SpiderFoot — автоматический OSINT-сканер

Техники поиска

Google Dorks — специальные операторы для точного поиска. Пример: site:example.com filetype:pdf

Поиск по социальным сетям, форумам, утекшим базам данных.

Этика

OSINT — это легально. Мы используем только открытые данные. Не путай с взломом.

⚔️ Пентестинг

Что такое пентест?

Penetration Testing — тестирование на проникновение. Легальная имитация атаки на систему для поиска уязвимостей.

Этапы пентеста

  1. Reconnaissance — сбор информации о цели
  2. Scanning — сканирование портов и сервисов
  3. Exploitation — эксплуатация уязвимостей
  4. Post-Exploitation — закрепление в системе
  5. Reporting — отчёт с рекомендациями

Инструменты

  • Nmap — сканер портов и сети
  • Metasploit — фреймворк для эксплуатации
  • Burp Suite — анализ веб-трафика
  • Hydra — брутфорс паролей
  • Wireshark — анализ сетевого трафика

🪟 Windows

Windows для безопасности

  • PowerShell — автоматизация, управление системой
  • Active Directory — управление пользователями и политиками

🛡️ Кибербезопасность

Основы безопасности

  • VPN — шифрование трафика, смена IP
  • 2FA — двухфакторная аутентификация везде
  • Менеджер паролей — не храни пароли в браузере
  • Шифрование — GPG для email, LUKS для диска
  • Анонимность — Tor, приватные браузеры, метаданные

Принцип минимума

Минимум прав — минимум риска. Не работай под root. Не давай приложениям лишних разрешений.

Проверка на утечки

Проверяй свои email на haveibeenpwned.com. Если данные утекли — меняй пароли немедленно.
